Blockify
========

This module exposes a number of core Drupal elements as blocks.

The currently supported elements are:

* Logo
* Site name
* Site slogan
* Mission
* Footer message
* Page title
* Breadcrumb
* Help
* Tabs
* Messages (errors)
* Messages (warning)
* Messages (status)
* Messages (all)
* Node (includes node and author objects)
